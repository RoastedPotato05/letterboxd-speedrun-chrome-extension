# Letterboxd Speedrun Chrome Extension #
  
## To run:  

1. Download zip to a directory and unpack

2. In Chrome, go to your extensions and turn on 'Developer mode' at the top right corner

3. Click the 'Load unpacked' button that should appear at the top left after turning on developer mode

4. Select the folder that you unpacked the code into

5. Open a tab to letterboxd.com and run the extension
